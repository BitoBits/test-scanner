# Vercel Scanner

A real-time Binance Perpetuals scanner that monitors volume, candle size, and open interest changes across multiple timeframes.

## Features

- **Real-time Monitoring**: Live alerts for top 5 volume and candle size events
- **Open Interest Tracking**: Tracks OI changes in USDT, coin value, and percentage
- **Data Explorer**: Browse historical top 5 data for any coin
- **Export/Import**: Backup and restore your data
- **Gap Filling**: Automatically fill missing data gaps
- **Multi-timeframe**: Monitors 15m, 1h, 4h, 12h, and 1d timeframes

## Usage

Simply open `index.html` in your browser. No build process required.

## Data Storage

All data is stored in browser localStorage. Use the Export/Import features to backup your data.
